To classify a list of locations as heterozygous sites or systematic errors prepare the input: 

==========================================================================


(*** see example run at the bottom ***)


command:   perl SysCall.pl   list_locations_to_classify    reads_file.sam    coverage_file    outfile   code_dir




Input:    (1)  list_locations_to_classify   (2)  reads_file.sam     (3)   coverage_file   (4) outfile    (5) code_dir


(1)   list_locations_to_classify  

This file is space delimited and holds in each row a position to classify.  It also holds the nucleotides present around the location specified, 2 bases in each direction. 

Some example lines are:


chr1 1840748 G G T A A

chr1 17113024 G G A G C

chr1 2467381 G G T C A

chr1 17535338 T A C T T



(2) reads_file.sam

This file contains the sequenced reads of the experiment in sam format. Notice that the algorithm makes use of the location, sequence and quality scores at the reads.

(3)   coverage_file 

This is a file holding the per-site coverage which is most similar to that of your data. It should be taken from the directory "coverage_files" and is used for the training step of the algorithm. 



(4) outfile 

The name of your outfile, to which different extentions will be attached for the temporary files and for the final two outfiles listing systematic errors and heterozygous sites. 


(5) code_dir

The directory in which the code is - the path you need to use to call SysCall.pl .




output:      (1)    list_heterozygous     (2)  list_sys_errors


(1) outfile.sys_errors    

Will hold the lines from input file "list_locations_to_classify" that have been classified as systematic errors. 


(2) outfile.heterozygous

Will hold the lines from input file "list_locations_to_classify" that have been classified as heterozygous sites. 



temporary  files:


(1) outfile.reads_parsed

holds the for every read that overlaps a position in the file list_locations_to_classify its parsed features. 


(2) outfile.table_all

holds for each location in file list_locations_to_classify the different features for both directions


(3) outfile.table_chosen

holds for each location the features chosen


(4) outfile.tmp1

holds the temporary files for the output.


(5) outfile.tmp2

holds the temporary files for the output. 


Example run:


from directory Example/  run:


perl ../SysCall.pl   list_locations  fastq_reads_chr19.sam  ../Coverage_Files   list_locations_out  ../


output:

-- the list of locations classified as systematic errors will be in file:   list_locations_out.sys_errors

-- the list of locations classified as heterozygous will be in file:   list_locations_out.heterozygous
